Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM. We can select an element in DOM : by Tag Name, by Class Name, by ID, by querySelector and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML : innerHTML gets and sets html elements(tags).
innerText : innerText is the visible text inside of an element.
textContent : textContent gets and sets visible and hidden both text inside of an element.

Question-3: What is event delegation in the DOM?

Event delegation is a technique. By using this technique we can give a single event listener to a parent element and it allows to handle events of its child elements. We don't need to give each child separate event listener.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event starts from the target element and propagates up through its parent elements until it reaches the root

Question-5: How do you create, add, and remove elements using JavaScript?

Create element : We can create an element by using createElement() method.
Add element : We can add element by using append() and appendChild() method.
Remove element : We can remove an element by using removeChild(), remove() method.
